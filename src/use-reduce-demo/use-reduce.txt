useReducer 훅 : reducer 함수와 초기 상태 받는 훅
reducer 함수가 작동하여 새 상태를 반환할 경우 리렌더링 유발 및 새 상태를 반환해주는 역할을 수행

reducer 함수 : 상태랑 액션(수행할 작업 정보가 담긴 객체)를 받아서 새 상태를 반환하는 함수(단 변경사항이 없으면 기존 상태 반환 가능)
액션 : dispatch 함수를 통해 reducer 함수로 전달할 작업에 대한 내용이 담긴 객체

액션 상수 : 액션의 type 문자열 값을 상수화(오타 방지 및 명령어 문자열 변경 가능 용이)

액션 생성 함수 : 액션을 매번 만들기 않고 함수 호출을 통해서 액션 객체를 반환해주는 함수(실수 방지 및 코드 가독성 향상)
